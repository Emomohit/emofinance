# EmoFinance — AI Finance Copilot 💰

A stunning, offline-first, modern finance tracking web app built with Vanilla JS, HTML, and CSS.

## Features
- **Frontend Only:** Runs entirely in the browser with `localStorage`. No backend required.
- **AI Insights:** Uses a local rule-engine to generate financial tips based on your transactions.
- **Glassmorphism UI:** Built with premium, ultra-modern CSS techniques.
- **Charts:** Visualizes cash flow and expenses using Chart.js.

## Getting Started
1. Clone the repository.
2. Open `index.html` in your browser.
3. Start logging your expenses!
