      About EmoFinance

      EmoFinance is a single-file, offline-friendly finance tracker with a built-in AI copilot. When previewed inside Claude, AI insights, chat and auto-categorization call Claude live. When self-hosted (e.g. GitHub Pages) without a connected API key, those same features fall back to a local rule-based analysis engine automatically — so the app always works. No backend, no build step, no tracking.
